# 配色方案提取

## 来源：两份PDF（瑶光智控矿山方案 + 智慧供热方案）

### 核心配色
- 背景：纯白 #FFFFFF，浅灰 #F8F9FA / #F5F5F7
- 主色（标题/强调）：深蓝紫 #3B3FD8 / #4040C8（类似 indigo/blue-violet）
- 次主色（卡片/图标背景）：中蓝紫 #5B5FE8 / #6366F1
- 浅紫色块（highlight box）：淡紫 #E8E8F8 / #EBEBFA
- 供热系统图节点：蓝紫渐变 #5B5FE8 → #8B8FFF
- 换热站节点：浅紫 #A8AAFF / #C4C5FF
- 管道颜色：中灰 #9CA3AF（供回水管）+ 深紫阀门 #6366F1
- 文字主色：深灰 #1F2937
- 文字次色：中灰 #6B7280
- 表格行交替：白 / 浅灰 #F9FAFB
- 强调色（橙/红）：#EF4444（预警）

### 字体风格
- 标题：粗体，蓝紫色 #3B3FD8
- 正文：常规，深灰 #374151
- 标签/badge：圆角矩形，浅紫底+深紫字

### 设计语言
- 白底为主，大量留白
- 卡片用浅灰或白色，细边框
- 主色调：蓝紫（indigo）系
- 强调色：橙红用于预警
- 图表/节点：蓝紫渐变填充
- 整体感：干净、专业、科技感但不暗黑
